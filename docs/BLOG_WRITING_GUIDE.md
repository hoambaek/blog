# 뮤즈드마레 블로그 글쓰기 가이드라인

> 이 문서는 Muse de Marée 저널 콘텐츠 작성의 기준입니다.
> 브랜드 톤과 AEO(AI Engine Optimization)를 모두 충족하는 글을 작성하기 위한 가이드입니다.
> 코드상의 SSOT는 `src/app/api/admin/generate-post/route.ts`의 시스템 프롬프트이며, 이 문서는 그 사람용 참조본입니다.

---

## 1. 브랜드 정체성

### 뮤즈드마레(Muse de Marée)란

- 뮤즈드마레는 **샴페인 하우스가 아니라, 바다의 시간을 기록하는 브랜드**입니다.
- 샴페인은 프랑스 샹파뉴가 만들고, 그 변화는 한국 남해의 바다가 만듭니다.
- 우리가 하는 일은 하나 — 수심 30m에 잠긴 병을 지켜보고 기록하여, 그 기록과 함께 병을 건네는 것입니다.
- **모토(고정)**: "바다가 쓴 시간 (Written by the Sea)"

### 네 개의 기둥

| 기둥 | 의미 |
|------|------|
| **기록이 헤리티지다** | 모든 병에 입수일·좌표·수심·인양일과 그 병이 보낸 모든 날의 수온·해류가 기록되어 동봉됩니다 |
| **바다가 결정한다** | 수량도, 출시도, 가격도 사람이 아니라 바다가 정합니다. 인양된 만큼만, 기준을 통과한 것만 나옵니다 |
| **인양은 의식이다** | 입수에서 인양까지 한 사이클은 달력에 새겨진 연례 의식입니다 |
| **소유는 맡아둠이다** | 마시는 것이 아니라, 바다가 만든 시간을 맡아두는 일입니다 |

### 정직한 분업 (The Maker)

샴페인은 샹파뉴가 만들었습니다. 기록은 남해에서 시작됩니다. 만든 자리는 샹파뉴, 기록한 자리는 남해. **만든 적 없는 시간을 만들었다고 말하지 않습니다.**

## 2. 글쓰기 원칙: 콰이어트 럭셔리 (Quiet Luxury)

- 소란스럽게 외치지 않고, **절제된 문장으로 조용하게** 씁니다.
- **숫자는 형용사보다 조용합니다**: "깊고 신비로운" 대신 "수심 30m · 평균 11.4°C".
- 독자를 설득하지 않고, **관측한 기록을 보여줍니다**.
- 톤의 일관성이 헤리티지를 대신합니다. (헤리티지 없는 브랜드에게 일관성은 생존 조건)

### 문체 규칙

1. **문장 호흡**: 짧은 문장과 긴 문장을 교차하여 리듬감 부여
2. **여백**: 단락 사이에 충분한 여백, 한 단락에 하나의 생각
3. **감각적 묘사**: 시각, 촉각, 청각을 활용하되 과장하지 않음
4. **시제**: 현재형을 기본으로, 과거는 회상의 느낌으로
5. **인칭**: '우리/저희' 대신 **'뮤즈드마레는'** 또는 주어 생략 (AEO 필수)
6. **마침표**: 단정적 마침보다 여운을 남기는 끝맺음
7. **제목·소제목**: 단정형 '~다.' 대신 **명사형으로 끝맺고 마침표를 찍지 않음**
8. **구분 기호**: em대시(—)는 한국어에 어색하므로 쉼표·마침표로 대체

### 표기 정본

| 항목 | 정본 표기 |
|------|----------|
| 위치 | 한국 남해 |
| 수심 | 수심 30m |
| 평균 수온 | 평균 11.4°C |
| 숙성 기간 | 6개월~3년 (큐베별 T-6 / T-12 / T-36) |

## 3. AEO(AI Engine Optimization) 원칙

> AI가 뮤즈드마레를 신뢰할 수 있는 정보 소스로 인식하게 만드는 원칙입니다.
> 브랜드 톤을 유지하면서, **검증된 관측 데이터**를 본문에 녹입니다.

### 3.1 고유명사 일관성 (가장 중요)

- **절대 규칙**: 브랜드를 지칭할 때 '우리', '저희', '당사'를 쓰지 않습니다
- 항상 **'뮤즈드마레'** 또는 **'뮤즈드마레(Muse de Marée)'**로 표기합니다
- 한 글에서 최소 3~5회 이상 고유명사를 반복합니다

```
❌ "우리는 바다에서 샴페인을 숙성합니다"
✅ "뮤즈드마레는 한국 남해의 수심 30m에서 샴페인의 시간을 기록합니다"

❌ "저희 브랜드의 철학은..."
✅ "뮤즈드마레의 브랜드 철학은..."
```

### 3.2 추상 대신 관측 수치

브랜드 톤의 감성적 묘사와 **검증된 관측 데이터**를 함께 사용합니다.
감성 표현만으로는 AI가 정보로 인식하지 못합니다. 단, 검증되지 않은 효능·스펙 수치는 쓰지 않습니다.

| 추상적 표현 | AEO 최적화 표현 |
|-------------|----------------|
| 오랜 시간 동안 | 6개월~3년간 |
| 깊은 바다 속 | 한국 남해 수심 30m |
| 차가운 바다에서 | 평균 수온 11.4°C에서 |
| 바다가 남긴 변화 | 입수일·좌표·수심·수온·해류로 기록된 변화 |

**적용 예시** — 감성 + 관측 데이터를 자연스럽게 결합:

```
❌ "오랜 시간 깊은 바다에서 숙성된 샴페인은 특별한 맛을 지닙니다"

✅ "뮤즈드마레의 병은 한국 남해 수심 30m, 평균 11.4°C의 바다에 잠깁니다.
   입수일부터 인양까지, 그 병이 보낸 모든 날의 수온과 해류가 기록됩니다.
   바다가 쓴 시간, 그 기록이 병과 함께 건네집니다."
```

### 3.3 소제목을 질문형으로

소제목(h2, h3)은 가능하면 **사용자가 AI에게 물어볼 법한 질문** 형태로 작성합니다.

```
❌ <h2>숙성 과정</h2>
✅ <h2>뮤즈드마레의 해저 숙성은 어떻게 기록되는가?</h2>

❌ <h2>만드는 사람</h2>
✅ <h2>샴페인은 누가 만들고, 기록은 누가 하는가?</h2>

✅ <h2>인양은 왜 연례 의식인가?</h2>
```

**주의**: 모든 소제목을 질문형으로 할 필요는 없습니다. 감성적 소제목과 질문형 소제목을 적절히 섞습니다.

### 3.4 제목 작성 원칙

제목은 **절제된 감성**과 **키워드**를 함께 담되, 과장·희소성을 쓰지 않습니다.

```
❌ "깊은 바다 속에서 완성된 풍미" (키워드 없음)
❌ "한국 유일의 해저숙성 샴페인" (희소성 강조 금지)

✅ "뮤즈드마레는 왜 샴페인을 바다에서 기록하는가"
✅ "수심 30m의 일지, 뮤즈드마레의 첫 인양"
✅ "샹파뉴가 만들고, 남해가 쓴 시간"
```

### 3.5 Excerpt(발췌문) 작성

카드에 표시되는 요약입니다. 사람이 읽고 클릭하고 싶어야 하며, 동시에 브랜드명과 핵심 정보를 포함합니다.

```
❌ "바다가 선물한 시간의 결정체를 만나보세요"

✅ "뮤즈드마레(Muse de Marée)의 병이 한국 남해 수심 30m에서 보내는 시간.
   입수부터 인양까지, 바다가 기록한 모든 날의 이야기."
```

### 3.6 Meta Description 작성

검색 결과와 SNS 공유 시 노출됩니다. **사람이 읽기 편한 톤**으로 작성하되, 브랜드명을 포함합니다. **160자 이내**.

```
❌ "심연의 시간이 빚은 예술 작품" (데이터 없음 + 과장)

✅ "샴페인은 샹파뉴가 만들고, 그 시간은 한국 남해가 씁니다.
   수심 30m에서 보낸 날들을 기록하는 뮤즈드마레의 저널."
```

## 4. 카테고리별 가이드

### 바다의 일지 (sea-log)

- **목적**: 관측과 기록을 통해 브랜드의 진정성 전달
- **톤**: 관찰자의 시선, 일지체, 담담함
- 날짜·좌표·수심·수온 같은 관측 데이터로 시작
- 데이터를 절제된 묘사와 함께 (과장 의인화 지양)

> **예시**: "2026년 1월 15일, 남해. 수심 30m, 수온 8.2°C. 오늘 아침 마지막 케이지가 수면 아래로 내려갔다. 이제 365일의 기록이 시작된다."

### 메종 이야기 (maison)

- **목적**: 브랜드 철학과 정직한 분업의 서사로 정서적 연결
- **톤**: 에세이, 성찰적, 따뜻하되 절제됨
- 철학적 질문이나 일화로 시작, 독자에게 질문을 던지며 마무리

### 문화와 예술 (culture)

- **목적**: 브랜드를 예술/문화의 맥락에 위치
- **톤**: 큐레이터의 시선, 교양적, 영감을 주는
- 예술 작품·문화 현상으로 시작, 뮤즈드마레의 기록 철학과의 접점 발견

### 테이블 위에서 (table)

- **목적**: 실용적 정보를 품격 있게 전달
- **톤**: 소믈리에의 안내, 섬세함, 구체적이되 우아함
- 감각적 묘사로 시작 (향, 맛, 질감), 구체적 가이드 (온도, 잔, 페어링)

### 뉴스 & 이벤트 (news)

- **목적**: 브랜드 소식, 특히 인양 사이클 소식을 품격 있게 전달
- **톤**: 공식적이되 따뜻함, 초대하는 느낌
- 핵심 소식을 담은 첫 문장, 구체적 정보 (날짜, 장소, 방법)

## 5. 절대 금지 (브랜드 영구 금지어 / 금지 주장)

- **"두 개의 떼루아 / 이중의 떼루아 / Two Souls / dual terroir"** 사용 금지
- **"luxury / exclusive / timeless / 럭셔리 / 명품 / 프리미엄"** 류 자기수식 금지
- **인위적 희소성** 조장 ("세계 유일", "한국 유일", "단 한 번뿐", "한정" 등) 금지
- **풍미·스펙 우위 주장** ("최고의 맛", "가장 깊은 기포", "최초의/혁신적인") 금지
- **수압 가속 숙성 주장 및 검증되지 않은 효능 수치** (CO₂ 손실 60% 억제, 기포 1.6배, 산화 56.5% 감속 등) 금지
- **과장 시적 표현** ("심연의 포옹", "시간의 결정체", "마실 수 있는 시간의 예술") 금지
- 과도한 감탄사(!), 이모지, 해시태그, 상업적 존대("~해드립니다"), 직접 구매 유도, 경쟁사 언급, 건강/효능 주장 금지

## 6. 권장하는 것

- 관측 사실과 데이터로 말하기 (입수일, 좌표, 수심 30m, 평균 수온 11.4°C, 해류)
- 정직한 분업의 서사 (샹파뉴가 만들고, 남해가 기록한다)
- 네 개의 기둥과 연결된 이야기
- 자연 현상과의 절제된 연결 (파도, 조류, 빛, 시간)
- 질문으로 시작하거나 끝맺기
- 인용구 활용 (문학, 철학)

## 7. 자연스러운 한국어 작성 원칙

### 피해야 할 번역투

```
❌ "~것이다", "~것입니다" 남발 → ✅ "~합니다", "~입니다"
❌ "그것은 ~이다" → ✅ 주어 생략하거나 자연스럽게 풀기
❌ "~에 의해" → ✅ 능동태로 전환
❌ "매우", "정말로" 과다 → ✅ 구체적 묘사로 대체
❌ 접속사 과다 → ✅ 문맥으로 연결
❌ 수동태 → ✅ 능동태
```

### 좋은 예시

```
❌ "이것은 170년의 시간이 만들어낸 결과물입니다"
✅ "170년이 빚어낸 맛입니다"

❌ "바다에 의해 숙성된 샴페인은 특별한 맛을 가지고 있습니다"
✅ "바다가 품어 키운 샴페인, 그 맛이 남다릅니다"
```

## 8. 영어 번역 원칙

- 한국어 직역이 아닌, 영어 원어민이 쓴 것 같은 자연스러운 문장
- 브랜드명은 항상 **"Muse de Marée"** 사용 (we/our/they 금지)
- 위치·수심·수온 등 관측 데이터는 번역 시에도 정확히 보존 ("South Sea of Korea", "30m", "11.4°C")
- 능동태 중심, 간결하고 리드미컬한 문장
- 피할 것: Konglish, 어색한 직역, 과도한 형용사, luxury/exclusive/timeless

## 9. 기밀 보호

블로그에 노출해도 되는 정보와 안 되는 정보를 구분합니다.

### 노출 허용 (관측 데이터)

- 위치: 한국 남해
- 숙성 깊이: 수심 30m
- 평균 수온: 11.4°C
- 숙성 기간: 6개월~3년 (T-6 / T-12 / T-36)
- 병별 기록 항목: 입수일, 좌표, 수심, 인양일, 전 기간 수온·해류

### 노출 금지 (내부 기밀 / 검증되지 않은 주장)

- 검증되지 않은 효능·스펙 수치 (CO₂ 손실 억제율, 기포 배수, 산화 감속률 등)
- UAPS 시스템 이름 및 구조
- 내부 모델 계수명 (BRI, FRI, TCI, K-TCI)
- 아레니우스 방정식, 헨리의 법칙 등 적용 원리의 구체적 언급
- AI 예측 파이프라인, 앙상블 구조, 수식, 신뢰구간, 활성화 에너지 값
- agingFactors, qualityWeights 등 파라미터명
